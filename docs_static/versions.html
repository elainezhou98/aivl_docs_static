GLOTRA 操作指南 · 版本入口

下列为同一套页面的多个 HTML 文件，共用 assets/ 与 manual-config.js；通过根元素 data-manual-variant 区分 includeAI、platformMarket、procurementAnalyticsTools （进货中心 · 市场分析 / 选品对比 / 仪表盘及下属小节）。打开任一项即可在浏览器中直接预览对应版本。

  • full.html 版本 1（full）：有 AI、有平台市场；无市场分析 / 选品对比 / 仪表盘。
  • ai-market.html 版本 2（ai-market）：有 AI、有平台市场；无市场分析 / 选品对比 / 仪表盘。
  • ai-no-market.html 版本 3（ai-no-market）：有 AI、无平台市场；无市场分析 / 选品对比 / 仪表盘。
  • no-ai-no-market.html 版本 4（no-ai-no-market）：无 AI、无平台市场；有市场分析 / 选品对比 / 仪表盘。
  • no-ai-market.html 版本 5（no-ai-market）：无 AI、有平台市场；有市场分析 / 选品对比 / 仪表盘。
